# Face_A7

## 说明
人脸识别项目A7组的代码，本项目人脸检测使用了faceboxes，人脸识别使用的是sphereface。训练所使用的数据集都为公开数据集：umdfaces、webface和lfw

主要有以下几个文件夹
- CaffeOnACL：添加了人脸检测需要layer，其实就是SSD的几个layer
- face-demo：根据官方的代码改的demo，添加了我使用的方法总共两个类
- test-script:测试模型的文件
- train-script:训练模型的文件
